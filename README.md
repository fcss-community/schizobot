# Информация
Бот для бесед ВК генерирующий сообщения.

# Использование
На данный момент можно вызвать генерацию только командой формата:
```
schizo <текст>
```

# Установка
```sh
$ git clone https://github.com/fcss-community/schizobot.git
$ cd schizobot
$ pip3 install -r requirements.txt
```

# Запуск
```sh
$ python3 main.py
```
